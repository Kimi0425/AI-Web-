# 冒泡排序算法项目

这是一个完整的冒泡排序算法实现项目，专为PyCharm开发环境设计。

## 🎯 项目特点

- **基础冒泡排序**：经典实现，易于理解
- **优化版本**：双向冒泡排序（鸡尾酒排序）
- **可视化功能**：显示排序过程的每一步
- **交互模式**：用户可输入自定义数据
- **性能统计**：比较次数、交换次数、执行时间
- **PyCharm集成**：完整的IDE配置

## 🚀 快速开始

### 1. 打开项目
在PyCharm中打开 `bubble_sort_project` 文件夹

### 2. 运行方式

#### 基础运行
```bash
python bubble_sort.py
```

#### 演示模式
```bash
python bubble_sort.py demo
```

#### 交互模式
```bash
python bubble_sort.py interactive
```

### 3. PyCharm运行配置
项目已配置3个运行配置：
- `bubble_sort` - 基础运行
- `bubble_sort_demo` - 演示模式
- `bubble_sort_interactive` - 交互模式

## 📊 功能说明

### BubbleSortVisualizer类
- `basic_bubble_sort()`: 基础冒泡排序
- `optimized_bubble_sort()`: 优化版双向排序
- `generate_test_data()`: 生成测试数据
- `print_statistics()`: 打印统计信息

### 测试用例
项目包含6个测试用例：
1. 常规乱序数组
2. 小数组
3. 已排序数组
4. 逆序数组
5. 空数组
6. 单元素数组

## 📈 性能对比

| 算法版本 | 时间复杂度 | 空间复杂度 | 特点 |
|---------|------------|------------|------|
| 基础冒泡 | O(n²) | O(1) | 简单易懂 |
| 优化版本 | O(n²) | O(1) | 提前终止，双向扫描 |

- **Python版本**: 3.6+

## 📝 使用示例

```python
from bubble_sort import BubbleSortVisualizer

# 创建实例
sorter = BubbleSortVisualizer()

# 生成测试数据
data = [64, 34, 25, 12, 22, 11, 90]

# 执行排序
sorted_data, stats = sorter.basic_bubble_sort(data)

# 查看结果
print("排序结果:", sorted_data)
sorter.print_statistics(stats)
```
